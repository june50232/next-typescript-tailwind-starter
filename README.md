## Usage

This project using node >= 12.13.0 & yarn

#### Docker Compose Hot Reload

```bash
docker-compose up -d --build
```

#### Docker Debug

```bash
docker create --name="tmp_$$" image:tag
docker export tmp_$$ | tar t
docker rm tmp_$$
```
